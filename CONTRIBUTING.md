# Contributing

Thank you for your interest in VRAXIA.

---

## This Repository

This is a **portfolio and showcase repository**. It contains conceptual documentation, architecture descriptions, and platform overviews.

**The production codebase is private.**

---

## What You Can Contribute

### Feedback and Discussion

Open a [GitHub Discussion](../../discussions) to:

- Ask questions about the architecture or concepts
- Discuss AI system design patterns
- Share use cases or industry applications
- Suggest documentation improvements

### Issues

Use GitHub Issues for:

- Documentation errors or typos
- Unclear explanations in architecture or concept docs
- Broken links or formatting problems

→ Use the issue templates provided in `.github/ISSUE_TEMPLATE/`

### What We Do Not Accept

- Pull requests with application code
- Backend implementations
- API integrations
- Database schemas
- Any production system components
